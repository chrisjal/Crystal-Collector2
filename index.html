Crystal Collector

Click to view/hide instructions

You will be given a random number goal at the start of the game. Your objective is to match this number exactly by using the value of the four crystals below.

Clicking each crystal will add a specific value to your current total. These values will continue to add up until you meet your goal (a win) or exceed your goal (a loss).

The value of each crystal is hidden until it is clicked.

When the game ends, the value of the crystals and goal will change!
